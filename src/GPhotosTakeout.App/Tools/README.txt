הנח כאן את exiftool.exe (מ-https://exiftool.org).

הורד את "Windows Executable", חלץ, ושנה את שם הקובץ מ-"exiftool(-k).exe" ל-"exiftool.exe",
והנח אותו בתיקייה זו. האפליקציה תזהה אותו אוטומטית ותכתוב metadata לתוך הקבצים.

ללא הקובץ הזה, האפליקציה עדיין מארגנת ומתארכת את התמונות — אך לא כותבת EXIF/XMP.
